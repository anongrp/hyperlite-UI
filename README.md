## <p align='center'> Hyperlite-UI </p>
------

#### Setup
```bash
> git clone https://github.com/anongrp/hyperlite-UI.git
> cd hyperlite-UI
```


##### For Dev
```bash
$ npm run dev
```

##### For Prod
```bash
$ npm start
```

**<p align='center'>&<p>**

##### To Deploy
```bash
$ npm run deploy
```
